Copy skill files from `docs/skills/` into your AI tool's commands directory:

```bash

cp docs/skills/explain-abap-code.md ~/.claude/commands/
cp docs/skills/fix-abap-code.md ~/.claude/commands/
# ... or copy all
cp docs/skills/*.md ~/.claude/commands/
```
